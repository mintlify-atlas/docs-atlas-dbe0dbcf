---
title: "Get User Details"
api: "GET /api/user/detail/{id}"
description: "Retrieve detailed information about a specific user by their ID"
---

## Endpoint

```
GET /api/user/detail/{id}
```

This endpoint retrieves detailed information about a user. No authentication is required.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the user to retrieve
</ParamField>

## Response

<ResponseField name="code" type="integer">
  HTTP status code (200 for success, 404 for not found)
</ResponseField>

<ResponseField name="status" type="string">
  Status of the request: `success` or `error`
</ResponseField>

<ResponseField name="user" type="object">
  The user object containing all user details
  
  <ResponseField name="id" type="integer">
    User's unique identifier
  </ResponseField>
  
  <ResponseField name="name" type="string">
    User's first name
  </ResponseField>
  
  <ResponseField name="surname" type="string">
    User's last name
  </ResponseField>
  
  <ResponseField name="email" type="string">
    User's email address
  </ResponseField>
  
  <ResponseField name="role" type="string">
    User's role (e.g., `ROLE_USER`)
  </ResponseField>
  
  <ResponseField name="description" type="string">
    User's profile description (optional)
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    Timestamp when the user was created
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    Timestamp when the user was last updated
  </ResponseField>
</ResponseField>

<ResponseField name="message" type="string">
  Error message (only present when status is `error`)
</ResponseField>

## Request Example

<CodeGroup>

```bash cURL
curl -X GET "https://api.example.com/api/user/detail/1"
```

```javascript JavaScript
fetch('https://api.example.com/api/user/detail/1')
  .then(response => response.json())
  .then(data => console.log(data));
```

```php PHP
$userId = 1;
$response = file_get_contents("https://api.example.com/api/user/detail/{$userId}");
$data = json_decode($response, true);
```

</CodeGroup>

<Note>

</Note>
