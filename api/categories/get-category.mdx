---
title: 'Get Category'
description: 'Retrieve a single category by ID'
---

## Endpoint

<ParamField path="method" type="string" default="GET">
  GET
</ParamField>

<ParamField path="endpoint" type="string">
  /api/category/{id}
</ParamField>

<Note>
  This endpoint does not require authentication. It is publicly accessible.
</Note>

## Path Parameters

<ParamField path="id" type="number" required>
  The ID of the category to retrieve
</ParamField>

## Response

Returns a JSON object containing the requested category or an error if not found.

### Success Response

<ResponseField name="code" type="number">
  HTTP status code (200 for success)
</ResponseField>

<ResponseField name="status" type="string">
  Response status ('success')
</ResponseField>

<ResponseField name="category" type="object">
  The category object
  
  <ResponseField name="id" type="number">
    Category ID
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Category name
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    Timestamp when the category was created
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    Timestamp when the category was last updated
  </ResponseField>
</ResponseField>

### Error Response

<ResponseField name="code" type="number">
  HTTP status code (404 for not found)
</ResponseField>

<ResponseField name="status" type="string">
  Response status ('error')
</ResponseField>

<ResponseField name="message" type="string">
  Error message describing what went wrong
</ResponseField>

## Example Request

```bash cURL
curl -X GET http://your-domain.com/api/category/1
```

## Example Responses

### Success Response (200)

```json
{
  "code": 200,
  "status": "success",
  "category": {
    "id": 1,
    "name": "Technology",
    "created_at": "2024-01-15T10:30:00.000000Z",
    "updated_at": "2024-01-15T10:30:00.000000Z"
  }
}
```

### Error Response (404)

```json
{
  "code": 404,
  "status": "error",
  "message": "La categoria no existe."
}
```

## Implementation Details

This endpoint is defined in `CategoryController.php:30` and uses Laravel's `find()` method to retrieve a category by its ID. If the category is not found, it returns a 404 error with a Spanish message.
